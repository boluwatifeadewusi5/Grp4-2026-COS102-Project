Built For The Desktop Demo

The app keeps a simple Python codebase while demonstrating real workflows across three role types.

Public Users

Post community updates, like and comment, manage friends, send casual-only messages, and export summaries.

NGOs

Discover Government partners, request partnerships, submit agreements, upload documents, manage projects, and file reports.

Government

Review partner requests, approve or reject agreements, monitor project progress, and keep a searchable activity history.

Cloud-Ready Shared Data

The executable can run offline with SQLite or connect to a hosted Postgres database with one environment variable.

1

Create a hosted Postgres database

Supabase or Neon are good free prototype options. Copy the pooled connection string.

2

Set the database URL

Set CIVIC_CONNECT_DATABASE_URL before running or packaging the app.

3

Build and publish the EXE

Use PyInstaller locally or the included GitHub Actions workflow, then attach the EXE to a GitHub Release.
