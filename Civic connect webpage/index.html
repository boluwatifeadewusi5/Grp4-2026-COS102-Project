Built For Desktop Collaboration

The app keeps a simple Python codebase while showing real workflows across three role types.

Casual Users

Post community updates, like and comment, manage friends, follow NGO/Government public profiles, send casual-only messages, and export summaries.

NGOs

Discover organizations, request relations, submit agreements with Government partners, upload documents, manage projects, download project CSV files, and file reports.

Government

Review relation requests, approve or reject agreements, collaborate with Government or NGO peers, monitor project progress, and keep a searchable activity history.

Postgres Shared Data

The executable can run offline with local Postgres or connect to a hosted Postgres database with one environment variable.

1

Choose local or hosted Postgres

Use local Postgres for offline work, or Supabase/Neon when multiple installs should share data.

2

Set the database URL when hosted

Set CIVIC_CONNECT_DATABASE_URL for online mode, or use CIVIC_CONNECT_LOCAL_DATABASE_URL for a custom local server.

3

Build and publish the EXE

Use PyInstaller locally, attach the EXE to a GitHub Release, then point the download button at that release asset URL.
