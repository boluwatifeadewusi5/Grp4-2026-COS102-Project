Built For Desktop Collaboration

The app keeps a simple Python codebase while showing real workflows across three role types.

Casual Users

Post community updates, like and comment, manage friends, send casual-only messages, and export summaries.

NGOs

Discover Government partners, request partnerships, submit agreements, upload documents, manage projects, and file reports.

Government

Review partner requests, approve or reject agreements, monitor project progress, and keep a searchable activity history.

Postgres Data

The executable can use a local Postgres server offline or connect to hosted Postgres online.

1

Choose local or hosted Postgres

Use local Postgres for offline work, or Supabase/Neon when multiple installs should share data.

2

Set the database URL when hosted

Set CIVIC_CONNECT_DATABASE_URL for online mode, or use CIVIC_CONNECT_LOCAL_DATABASE_URL for a custom local server.

3

Refresh and publish

Set CIVIC_CONNECT_REFRESH_SECONDS from 1 to 5, then build and attach the EXE to a GitHub Release.

Role-Based Workspaces

Each role has a focused workspace so casual conversations, NGO operations, and Government approvals stay separate.

Community Space

Casual users handle posts, comments, friendships, private messages, profile updates, and activity exports.

NGO Workspace

NGOs discover partners, submit agreements, manage projects, upload documents, and publish reports.

Government Review

Government users approve partners, review agreement changes, monitor projects, and track public initiatives.

Civic Connect is a desktop app. The website hosts the showcase and download; the app itself runs on Windows.
